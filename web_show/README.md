# 点云3D框体编辑器

一个基于Web的点云3D框体编辑和可视化工具。

## 功能特性

- ✅ **PCD点云文件加载** - 支持标准PCD格式的点云文件
- ✅ **3D可视化** - 基于Three.js的流畅3D渲染
- ✅ **框体管理** - 添加、编辑、删除3D框体
- ✅ **批量导入** - 支持从文本批量导入框体数据
- ✅ **精确调整** - 支持±0.01的微调功能
- ✅ **导出功能** - 导出框体数据到TXT文件
- ✅ **交互控制** - 鼠标旋转、平移、缩放点云

## 使用方法

### 1. 上传点云文件
- 点击"上传PCD点云文件"按钮
- 选择你的PCD点云文件

### 2. 添加框体
有两种方式添加框体：

#### 方式A: 手动添加
- 点击"添加新框体"按钮
- 在参数列表中调整框体的位置、尺寸和角度

#### 方式B: 批量导入（推荐）
- 点击"批量导入"按钮
- 在弹出的对话框中粘贴框体数据
- 数据格式：每行一个框体，参数用空格分隔
- 参数顺序：`x y z w l h rt`
  - x, y, z: 框体中心坐标
  - w: 宽度
  - l: 长度
  - h: 高度
  - rt: 旋转角度（弧度）

**示例格式：**
```
34.56 0 -1 69.12 79.36 4 0
8.69702 -1.82145 -0.671802 1.20388 0.627663 1.8344 4.79124
40.1654 -3.74482 0.828855 0.799536 0.672789 1.77988 2.82616
```

### 3. 调整框体
- **手动输入**: 直接修改输入框中的数值
- **微调按钮**: 使用 +/- 按钮以0.01为单位微调
- **实时预览**: 修改后立即在3D场景中看到变化

### 4. 删除框体
- 点击框体右侧的"删除"按钮

### 5. 导出数据
- 点击"导出TXT"按钮
- 自动下载包含所有框体信息的文本文件

## 3D场景控制

- **左键拖动**: 旋转视角
- **右键拖动**: 平移视角
- **滚轮滚动**: 缩放场景

## 文件说明

- `index.html` - 主页面
- `app.js` - 主要逻辑代码
- `OrbitControls.js` - Three.js轨道控制器

## 使用示例

假设你有一个检测结果的TXT文件，内容如下：
```
8.69702 -1.82145 -0.671802 1.20388 0.627663 1.8344 4.79124
40.1654 -3.74482 0.828855 0.799536 0.672789 1.77988 2.82616
```

只需要：
1. 打开文件，复制所有内容
2. 在网页中点击"批量导入"
3. 粘贴复制的数据
4. 点击"导入"
5. 框体将自动显示在3D场景中

## 技术栈

- Three.js - 3D渲染引擎
- WebGL - 硬件加速渲染
- 纯JavaScript - 无需后端服务器
